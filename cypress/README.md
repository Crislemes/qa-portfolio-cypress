## 🚀 Tecnologias
- Cypress
- JavaScript

## 📌 Cenários cobertos

- Login com sucesso
- Validação de acesso

### 📝 Cadastro

- Validação de campos

### 🔗 API
- GET de usuários
- Validação de status e dados
